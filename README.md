This repository contains the final capstone project for Modern Data Science (SIG742), completed as a collaborative group assignment.

The project analyzed over **552k business reviews**, focusing on data cleaning, exploratory data analysis, forecasting, and strategic decision insights. The work demonstrates the application of scalable data processing and predictive analytics to real-world business data.

---

## Project Summary

This project involved:
- Large-scale dataset handling using **PySpark** and **Pandas**
- Exploratory analysis to identify trends and patterns in review behavior
- Time-series forecasting to analyze temporal dynamics and predict future behavior
- Model comparison using **ARIMA** and **LSTM**
- Insight-driven recommendations based on data-supported evidence
